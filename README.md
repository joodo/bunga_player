# Bunga Player

A video player allows users watching same video share progress.

## Run

Get all dependencies:

```
flutter pub get
```

Run:

```
flutter run
```

Deploy:

```
dart run bunga_player:deploy
```

## Update assets

### Generate app icon:

```
dart run flutter_launcher_icons
```

### Fetch emojis:

```
dart run bunga_player:fetch_emojis
```
